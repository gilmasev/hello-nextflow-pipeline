Through this Nextflow training I learned how to:

- describe and utilize core Nextflow components sufficient to build a simple multi-step workflow,
- describe next-step concepts like operators, channel factories,
- customize modules, params, containers, config, and profiles,
- launch a Nextflow workflow locally and on an High Performance Computing (HPC) system,
- find and interpret outputs (results) and log files generated by Nextflow (eg. .command.sh and resource allocation reports),
- troubleshoot basic issues.
